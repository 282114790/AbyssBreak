## 1. 核心乐趣与游戏性进化

### 核心乐趣点

| 乐趣维度 | 描述 |
|-----------|------|
| **技能组合的惊喜感** | 4 个主动技能槽 + 元素共鸣 + 协同触发，每一局的 Build 都不同。当两个技能意外触发协同时，满屏特效 + 全屏 Banner 带来强烈正反馈。 |
| **渐进的力量膨胀** | 从一个基础技能到四技能满槽 → 进化 → 终结技就绪，玩家的战斗力在 15 分钟内经历"挣扎→成型→碾压→再度挣扎"的完整心流曲线。 |
| **即时操作快感** | 闪避翻滚、终结技的手动释放、消耗道具的战术使用，将纯自动战斗升级为"半自动 + 关键决策"混合节奏。 |
| **Roguelite 重开动力** | 每局的随机选择（技能/被动/遗物/诅咒/路线）确保不重复，加上 Meta 成长树的永久提升，即使失败也能积累进度。 |
| **深渊层的长线挑战** | 20 层深渊修正提供持续几十小时的难度阶梯，最终层"真·深渊"是终极目标。 |

### 游戏性进化路线图

```
阶段 1（当前）：核心战斗循环
  ├── 波次生存 + 升级选择 + Boss 挑战
  ├── 技能槽位上限 → 选择/替换策略
  └── 3 职业 × 19 技能 × 14 协同 = 组合空间

阶段 2（规划中）：Build 深度
  ├── 技能进化分支（同一技能可进化为不同形态）
  ├── 元素精通效果完善（3 同元素触发更强效果）
  └── 角色专属进化路线

阶段 3（远景）：内容扩展
  ├── 更多角色（刺客、牧师、召唤师……）
  ├── 特殊地图事件（陷阱房、宝藏房、精英挑战）
  ├── 无尽深渊排行榜
  └── 赛季制：每赛季新增技能/遗物/深渊修正
```

### 如何持续玩下去

1. **解锁全角色**：通过累计局数和最高分解锁战士、猎手
2. **挑战更高难度**：普通 → 困难 → 深渊
3. **攀登深渊层**：每通关解锁下一层修正，直到第 20 层"真·深渊"
4. **收集 Build 图鉴**：发现 10 种预设 Build 组合获得成就
5. **点满 Meta 树**：用灵魂石解锁永久强化节点
6. **探索所有进化**：集齐 9 种技能进化形态
7. **全角色通关**：每个角色的独特机制创造不同的游戏体验

## 24. 可扩展性分析

### 高扩展性（⭐⭐⭐⭐⭐）— 纯数据驱动，新增只需字典

| 系统 | 扩展方式 |
|------|----------|
| 技能 | 继承 `SkillBase.gd` + 在 `Main.gd` 注册 `SkillData` |
| 被动 | 在 `Main.gd` 注册 `PassiveData` 字典 |
| 遗物 | 在 `RelicRegistry.gd` 添加字典 |
| 敌人 | 在 `WaveManager.gd` 的 `enemy_presets` 添加字典 |
| 协同 | 在 `SynergySystem.gd` 的 `SYNERGIES` 添加字典 |
| 诅咒 | 在 `UpgradeSystem.gd` 的 `curse_pool` 添加字典 |
| 进化 | 在 `UpgradeSystem.gd` 的 `evolve_table` 添加字典 |
| 角色 | 在 `CharacterRegistry.gd` 注册字典 |
| 地图 | 在 `MapRegistry.gd` 注册字典 |
| Build 图鉴 | 在 `BuildCodex.gd` 的 `BUILD_DEFS` 添加字典 |
| Meta 节点 | 在 `MetaProgress.gd` 的 `UNLOCK_NODES` 添加字典 |

### 中等扩展性（⭐⭐⭐⭐）— 需配套逻辑

| 系统 | 说明 |
|------|------|
| 深渊层修正 | 数据定义容易，但每层独特逻辑需额外代码 |
| Boss | 预设容易添加，但独特 AI 和攻击模式需大量开发 |
| 章节事件 | 需要设计事件框架后才能批量扩展 |
| 终结技 | 需为每个角色编写独立的释放逻辑和特效 |

### 架构优势

- **EventBus 信号解耦**：系统间通过信号通信，新增系统无需修改现有代码
- **数据与逻辑分离**：所有内容定义集中在 Registry/数据文件中
- **代码驱动 UI**：无 `.tscn` 依赖，所有 UI 均代码生成，便于动态扩展
- **对象池意识**：`WaveManager` 已有帧跳过策略，性能可控

### 优先扩展建议

1. **P0**：为雷系/毒系各新增 2 个技能，平衡元素生态
2. **P0**：修复玻璃炮诅咒 `damage_bonus` 未生效的 bug
3. **P1**：为剩余 10 个技能添加进化路线
4. **P1**：丰富 Boss AI（阶段切换、弹幕模式、范围技能）
5. **P2**：实装剩余 5 个深渊层的专属逻辑
6. **P2**：为地图添加 gameplay 差异（障碍物、陷阱、地形效果）
7. **P2**：角色专属终结技
8. **P3**：章节事件系统
9. **P3**：新角色（刺客、牧师、召唤师）
