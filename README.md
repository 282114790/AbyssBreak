## 🎮 核心玩法

**深渊突围**是一款俯视角 Roguelite 弹幕生存游戏，灵感源自《吸血鬼幸存者》，融合了主动技能操作与高密度 Build 构筑。

### 游戏循环（一局约 15–30 分钟）

```
选择角色 → 选择路线 → 生存 + 击杀 + 升级 → Boss战 → 死亡/胜利 → 获得魂石 → 下一局更强
```

1. **选路线** — 每局开始三选一，决定整局节奏
2. **自动攻击 + 主动释放** — 技能自动触发，Q/E手动施放主动技能
3. **升级** — 每升一级从5个选项选2个，含高风险高收益的诅咒选项
4. **遗物** — 每3波掉落，提供强力被动加成，是 Build 核心
5. **商人** — 每5波出现，用魂石买强化
6. **Boss战** — 三阶段Boss，死亡时有仪式感
7. **结算** — 查看本局完整 Build，魂石存入元进度

### 乐趣点

- 🧩 **无限组合** — 15种技能 × 20种遗物 × 6种技能协同，每局 Build 都不同
- ⚡ **手感爽快** — 暴击黄字飞起、白闪击打、屏幕震动、死亡粒子爆散
- 🎲 **风险决策** — 每次升级、每个随机事件都是压力选择
- 📈 **滚雪球** — 前期弱、后期每次升级都是质变时刻
- 🌑 **Boss节奏** — 三阶段变色+加速，有真实的压迫感高潮

## 🕹 操作方式

| 按键 | 功能 |
|------|------|
| `WASD` | 移动（有惯性手感） |
| `Space` | 翻滚闪避（0.25秒无敌帧，0.7秒冷却） |
| `Q` | 主动技能槽位 1（虚空裂缝） |
| `E` | 主动技能槽位 2（时间减速） |
| `ESC / P` | 暂停（可查看当前完整 Build） |

**技能自动向优先目标发射**：精英敌人 > 最低血量 > 最近敌人

### Boss：深渊魔王 💜
每波次循环出现，三阶段设计：
- **阶段1（HP > 66%）** — 正常状态
- **阶段2（HP ≤ 66%）⚡狂暴** — 速度×1.4，攻击间隔×0.7，变红，屏幕震动
- **阶段3（HP ≤ 33%）💀濒死狂怒** — 速度×1.6，持续颤抖，变紫，高频攻击

## 🗺 波次与关卡节奏

### 三路线选择（每局开始）
| 路线 | 特色 |
|------|------|
| 💎 遗物之路 | 遗物掉落×2，敌人数量+20% |
| ⚔ 鲜血之路 | 经验×1.5，敌人HP+30%，魂石+30% |
| 🧙 商旅之路 | 商人每3波出现（默认5波），起始魂石×1.5 |

### 波次结构
每循环由4阶段组成：
```
小兵波（3波）→ 精英波（警告+精英涌现）→ 喘息期（30秒无敌人）→ Boss波
```

### 30秒事件机制
每30秒自动触发一件事（轮转）：
- 精英敌人突袭
- 遗物掉落
- 治疗气泡出现
- 环境特效

### 随机事件（每90秒）
4种文字事件，二选一高风险/低风险选项：
- 🌑 深渊裂缝（遗物 vs AOE伤害）
- 🩸 血之契约（伤害+25% vs 失去30% HP）
- 🔮 魔法涌流（CD减少 vs 清屏+一格CD变10秒）
- 👻 迷途灵魂（经验暴增 vs 最大HP+20%但不能回血）

## 📈 升级系统

升级时从 **5个选项** 中选 **2个**，含以下类型：
- 技能解锁 / 现有技能升级
- 被动技能
- 诅咒选项（高风险高回报）：
  - 💀 生命诅咒 — HP-20%，伤害+40%
  - 🌀 速度诅咒 — 移动速度-30%，冷却时间-40%
  - ⏳ CD诅咒 — 冷却时间+50%，技能伤害+60%
  - 🌱 再生诅咒 — 停止所有回血，每秒技能伤害+5%

## 💾 元进度（跨局永久成长）

每局结束后获得 **💎 魂石**，用于永久解锁：
- 新角色
- 起始遗物
- 路线选项
- 难度设置

**历史记录**：历史最高波次、最高得分永久保存。

## 🔊 音效分层系统

4条独立音频总线，避免打击感混成一团：
- **Normal** — 普通攻击，音量随伤害值变化
- **Skill** — 技能释放（爆炸、范围技）
- **Crit** — 暴击专属，音调偏高，一听就知道暴了
- **Death** — 敌人死亡、玩家受伤

## 🔭 下一步优化方向（创作者视角）

### 🔴 高优先级（影响核心体验）

1. **音效资源** — 目前音效框架完整，但实际 `.wav` 文件为占位符。需录制/购买：普攻打击声、技能施放声、暴击声、Boss 吼叫声。**这是当前最影响游戏感的缺口。**

2. **技能视觉差异化** — 所有技能目前靠颜色区分，投射物形状相似。至少需要3-4种不同外形（圆形弹、扇形弹、光环、落地符文）。

3. **角色美术** — 战士/猎手的行走动画帧（`warrior_walk_sheet.png` / `hunter_walk_sheet.png`）路径存在但资源未创建，角色外观与法师相同。

4. **Boss 独特技能** — 当前 Boss 只是属性强化版，无独特攻击模式。建议第二阶段增加"范围冲刺"，第三阶段增加"召唤小怪"。

### 🟡 中优先级（提升留存）

5. **更多角色（4-6个）** — 目前3角色，差异度足够，但数量少。建议加"术士"（诅咒专精）和"圣骑士"（治疗+护盾流）。

6. **成就系统完善** — `AchievementSystem.gd` 框架已在，但条件检测和UI奖励尚未接入到结算流程。

7. **每日挑战** — `DailyChallenge.gd` 已存在，需接入种子随机和HUD入口。

8. **Build 多样性增强** — 当前遗物效果大多是数值加成，建议加2-3个"玩法改变"类史诗遗物（如"死亡爆炸"、"子弹反射"）。

9. **关卡差异化** — 所有局都在同一张地图。波次越高可以引入地图障碍物、狭窄走廊等。

### 🟢 低优先级（打磨）

10. **触屏/手柄支持** — 操作已解耦，加虚拟摇杆即可移植到手机。

11. **更多敌人种类** — 当前5种基础敌人。建议加"爆炸桶形怪"（死亡爆炸）、"治疗者"（给周围敌人回血）。

12. **结算分享图** — 结算界面自动截图功能（`DisplayServer.get_screen_image()`），一键分享 Build。

13. **Mod 支持** — 当前数据驱动架构天然支持，只需外放 `EnemyData`/`RelicData`/`SkillData` 的 JSON 加载入口。
